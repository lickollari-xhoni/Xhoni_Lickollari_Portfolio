# 🚀 Portfolio - Xhoni Lickollari

Benvenuto nel repository del mio portfolio personale! Questo progetto rappresenta la mia vetrina professionale, sviluppata per mostrare le mie competenze come Full-Stack Web Developer.

## 🛠 Tecnologie Utilizzate

Il sito è stato costruito con un focus sulla pulizia del codice, la performance e l'esperienza utente (UX).

- **Backend:** Laravel (PHP)
- **Frontend:** HTML5, CSS3, JavaScript (ES6)
- **Framework UI:** Bootstrap 5
- **Fonts:** Space Grotesk, Syne (via Google Fonts)
- **Controllo Versioni:** Git & GitHub

## ✨ Caratteristiche Principali

- **Design Responsive:** Ottimizzato per ogni dispositivo, dal desktop allo smartphone.
- **Menu Interattivo:** Navigazione in overlay personalizzata con animazioni fluide.
- **UX Curata:** Integrazione di elementi visivi dinamici e interazioni hover per una navigazione coinvolgente.
- **Struttura Modulare:** Sviluppato utilizzando le *partials* di Laravel per garantire un codice manutenibile e scalabile.
